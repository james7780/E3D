E3D Cross Platform 3D Library
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Copyright James Higgs 2006-2014

This 3D library manages the 3D objects in your 3D game scene. It is platform-independant and 
renderer-independant, and it is easy to add extra platforms and renderers.
